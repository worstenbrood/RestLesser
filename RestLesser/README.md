 # RestLesser

RestLesser is a rest/OData client with no dependencies except for Newtonsoft.Json (it supports custom data adapters).
It includes basic, token, header, url and OAuth authentication.
The OData client support select, filter, expand, top and orderby.
The filter support and, or, eq, gt, lt, ge, le, ne, in, contains, startswith, endswith, toupper, tolower and substring.
